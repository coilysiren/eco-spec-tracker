# eco-spec-tracker

Tracks every Eco player's jobs (professions/specialties) with `active / total` counts.

Deployed at [eco-jobs-tracker.coilysiren.me](https://eco-jobs-tracker.coilysiren.me).

## Quick start

```sh
make build-native
make run-native     # -> http://localhost:4000 with autoreload
```

See [AGENTS.md](AGENTS.md) for the full layout and design notes.
